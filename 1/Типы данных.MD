JavaScript имеет 8 типов данных: 7 примитивов и объекты. Примитивы неизменяемы и передаются по значению, а объекты — по ссылке, что влияет на сравнения.

## Примитивы

Примитивы — это базовые атомарные значения без методов (хотя временно "оборачиваются" в объекты при вызове свойств).

- `string`: `'hello'`.[^1]
- `number`: `42`, `3.14`, `NaN`, `Infinity`.
- `bigint`: `123n` (для больших чисел).
- `boolean`: `true`, `false`.
- `null`: намеренное пустое значение.
- `undefined`: значение по умолчанию.
- `symbol`: уникальный идентификатор, `Symbol('id')`.[^4]

Пример: `typeof 'hello' // "string"`, `typeof null // "object"` (историческая ошибка).[^2]

## Объекты

Объекты — коллекции свойств (ключ-значение), изменяемы, передаются по ссылке.
Включают: `{a: 1}`, `[1,2]`, `() => {}`, `new Date()`.
Пример:

```js
let obj = {name: 'Иван'};
let copy = obj;  // ссылка, не копия
obj.name = 'Петр';
console.log(copy.name);  // 'Петр'
```

## Сравнение (== vs ===)
`===` — строгое, сравнивает тип и значение, без преобразований.  
`==` — нестрогое, преобразует операнды (абсурдные правила Abstract Equality) [web:30].  

| Оператор | `0 == false` | `'' == 0` | `' ' == 0` | `[] == 0` | `null == undefined` |
|----------|--------------|-----------|-------------|------------|---------------------|
| `==`     | `true`      | `true`   | `false`    | `true`    | `true`             |
| `===`    | `false`     | `false`  | `false`    | `false`   | `false`            |[web:31]

Примеры:  
```js
console.log(1 == '1');     // true (string → number)
console.log(1 === '1');    // false (типы разные)
console.log(true == 1);    // true (boolean → number)
console.log(null == 0);    // false
console.log([] == 0);      // true ([] → '' → 0)
```

Всегда используйте `===` — `==` приводит к багам [web:36].
