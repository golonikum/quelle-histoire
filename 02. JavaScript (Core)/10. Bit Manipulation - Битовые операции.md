## 🔢 **Bit Manipulation в JavaScript — Битовые операции**

JavaScript работает с **32-битными целыми числами со знаком** для битовых операций. Числа автоматически конвертируются из IEEE 754 double.

### **🛠️ Основные операторы**[^1][^2]

| Оператор | Название             | Что делает                      |
| :------- | :------------------- | :------------------------------ |
| `&`      | **AND**              | 1 если **оба** бита = 1         |
| `\|`     | **OR**               | 1 если **хотя бы один** = 1     |
| `^`      | **XOR**              | 1 если **разные** биты          |
| `~`      | **NOT**              | Инверсия всех битов             |
| `<<`     | **Left Shift**       | Сдвиг влево (×2)                |
| `>>`     | **Right Shift**      | Сдвиг вправо (÷2)               |
| `>>>`    | **Zero Right Shift** | Сдвиг вправо, заполнение нулями |

### **🔢 Примеры (5 = 0101, 3 = 0011)**

```javascript
console.log(5 & 3); // 0101 & 0011 = 0001 = 1 ✅
console.log(5 | 3); // 0101 | 0011 = 0111 = 7 ✅
console.log(5 ^ 3); // 0101 ^ 0011 = 0110 = 6 ✅
console.log(~5); // ~0101 = ...1010 = -6 (двухдольная арифметика)
console.log(5 << 1); // 0101 → 1010 = 10 (×2)
console.log(5 >> 1); // 0101 → 0010 = 2 (÷2)
```

### **🎯 Практические применения (Middle+/Senior)**

#### **1. Проверка чётности (быстрее `% 2`)**

```javascript
function isEven(n) {
  return (n & 1) === 0; // Последний бит = 0
}
isEven(4); // true, isEven(5); // false ✅ O(1)
```

#### **2. Флаги/permissions (битовые маски)**

```javascript
const PERMISSIONS = {
  READ: 1, // 0001
  WRITE: 2, // 0010
  DELETE: 4, // 0100
  ADMIN: 8, // 1000
};

let userRole = PERMISSIONS.READ | PERMISSIONS.WRITE; // 0011

// Проверка прав
if (userRole & PERMISSIONS.WRITE) {
  console.log("Может писать"); // ✅
}

// Добавление права
userRole |= PERMISSIONS.DELETE; // 0111
```

#### **3. Swap переменных (без временной)**

```javascript
let a = 5,
  b = 3;
a ^= b; // a = 5^3 = 6
b ^= a; // b = 3^6 = 5
a ^= b; // a = 6^5 = 3
// a=3, b=5 ✅
```

#### **4. Math.abs() быстрее**

```javascript
function abs(n) {
  return (n ^ (n >> 31)) - (n >> 31); // Для 32-бит
}
```

### **🚀 Production примеры**

#### **Permissions система**

```javascript
class Permissions {
  static READ = 1 << 0; // 00000001
  static WRITE = 1 << 1; // 00000010
  static DELETE = 1 << 2; // 00000100

  hasPermission(role, permission) {
    return (role & permission) === permission;
  }

  addPermission(role, permission) {
    return role | permission;
  }
}
```

#### **Memory optimization — упаковка флагов**

```javascript
// Вместо 4 boolean → 1 число (32 флага!)
const flags = 0; // 000...0000

// Пакуем
flags |= 1 << 0; // flag0 = true
flags |= 1 << 3; // flag3 = true

// Распаковываем
const flag0 = !!(flags & (1 << 0)); // true
```

### **🔥 Собес-задачи**

**1. "Проверить power of 2":**

```javascript
const isPowerOfTwo = (n) => n > 0 && (n & (n - 1)) === 0;
// 8=1000, 7=0111 → 1000 & 0111 = 0 ✅
```

**2. "Найти уникальное число" (XOR всех):**

```javascript
const singleNumber = (arr) => arr.reduce((a, b) => a ^ b);
// 1^1^2^2^3 = 3 ✅
```

### **🎯 Senior-level знания**

```
✅ Флаговые маски вместо объектов {read: true}
✅ 1<<n вместо Math.pow(2,n)
✅ n & (n-1) === 0 для power of 2
✅ XOR для unique number поиска
```
