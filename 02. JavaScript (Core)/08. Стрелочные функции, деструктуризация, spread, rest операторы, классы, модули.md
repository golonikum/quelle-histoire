# ES6+ фичи: Стрелочные функции, деструктуризация, spread/rest операторы, классы, модули.

ES6+ ввёл мощные фичи для упрощения кода: стрелочные функции, деструктуризацию, spread/rest, классы и модули. Они делают JavaScript чище и функциональнее.

## Стрелочные функции

Стрелки (=>) — краткая запись функций без своего this, arguments или super. Синтаксис: `const sum = (a, b) => a + b;`. Идеальны для коллбэков: `array.map(item => item * 2);`. Не подходят как конструкторы.

## Деструктуризация

Извлекает значения из массивов/объектов: `const [a, b] = [1, 2];` или `const {name, age} = user;`. С дефолтами: `const {name = 'Anon'} = {};`. Полезно в параметрах: `function greet({name = 'User'}) {}`.

## Spread и Rest

**Spread** (...) разворачивает: `const newArr = [...arr, 4];` или `Math.max(...numbers);` — копирует/сливает. **Rest** собирает остатки: `function sum(a, b, ...rest) {}` или `const [a, b, ...others] = arr;`.

| Оператор | Контекст         | Пример               |
| :------- | :--------------- | :------------------- |
| Spread   | Массивы/объекты  | `[...arr1, ...arr2]` |
| Rest     | Функции/деструкт | `...args`, `...rest` |

## Классы

Синтаксический сахар над прототипами: `class User { constructor(name) { this.name = name; } greet() { return `Hi, \${this.name}`; } }`. Поддерживают extends, super, статические методы.

## Модули

import/export для разделения кода: `export const func = () => {};` и `import {func} from './module.js';`. Дефолт: `export default Class;` / `import MyClass from './';`. Строго синхронны, tree-shaking в бандлерах.
