# **Основы RxJS**

**RxJS** — библиотека для **реактивного программирования**. Работает с **потоками данных** (Observable) и **операторами** для их обработки.

## 🎯 **1. Observable — поток данных**

**Observable** — ленивая коллекция будущих значений, которая **не выполняется**, пока на неё не **подписались**.

```javascript
import { Observable, of } from "rxjs";

// Создание Observable
const numbers$ = of(1, 2, 3, 4, 5); // ✅ Ленивый!

// Подписка = выполнение
numbers$.subscribe({
  next: (value) => console.log(value), // 1 2 3 4 5
  error: (err) => console.error(err),
  complete: () => console.log("完了!"),
});
```

## 🛠️ **2. Основные операторы создания**

| Оператор                      | Что делает         | Пример                   |
| :---------------------------- | :----------------- | :----------------------- |
| `of(1, 2, 3)`                 | Массив значений    | `of(1, 2, 3)`            |
| `from([1, 2, 3])`             | Из массива/объекта | `from(array)`            |
| `interval(1000)`              | Каждую секунду     | `0, 1, 2...`             |
| `fromEvent(element, 'click')` | События DOM        | клики мыши               |
| `ajax('/api')`                | HTTP запрос        | `Promise` → `Observable` |

```javascript
// Примеры
interval(1000).subscribe(console.log); // 0, 1, 2... каждую секунду
fromEvent(document, "click").subscribe((e) => console.log(e)); // Клик события
```

## 🔥 **3. Операторы трансформации (pipe)**

**Всегда используй `.pipe()`** для цепочки операторов!

### **map** — преобразование

```javascript
const doubled$ = of(1, 2, 3).pipe(map((x) => x * 2));
doubled$.subscribe(console.log); // 2, 4, 6
```

### **filter** — фильтрация

```javascript
const evens$ = of(1, 2, 3, 4).pipe(filter((x) => x % 2 === 0));
evens$.subscribe(console.log); // 2, 4
```

### **tap** — побочные эффекты (логирование)

```javascript
of(1, 2, 3)
  .pipe(
    tap((x) => console.log("before:", x)),
    map((x) => x * 2),
    tap((x) => console.log("after:", x)),
  )
  .subscribe(console.log);
// before: 1, after: 2
// before: 2, after: 4
```

## 🎯 **4. Комбинирующие операторы**

### **merge** — параллельное выполнение

```javascript
const left$ = of(1, 2, 3);
const right$ = of(4, 5, 6);
merge(left$, right$).subscribe(console.log); // 1, 4, 2, 5, 3, 6
```

### **combineLatest** — последние значения

```javascript
const name$ = of("Никита");
const age$ = of(30);
combineLatest([name$, age$])
  .pipe(map(([name, age]) => `${name}, ${age} лет`))
  .subscribe(console.log); // "Никита, 30 лет"
```

### **forkJoin** — все завершатся

```javascript
forkJoin({
  users: ajax("/api/users"),
  posts: ajax("/api/posts"),
}).subscribe(({ users, posts }) => {
  console.log(users.data, posts.data);
});
```

## 🚀 **5. Операторы времени**

```javascript
// debounceTime — ждём паузу
fromEvent(input, 'input').pipe(
  map((e: any) => e.target.value),
  debounceTime(300),           // 300мс паузы
  distinctUntilChanged()       // Только новые значения
).subscribe(searchTerm => {
  console.log('Search:', searchTerm);
});
```

## 🧠 **6. Операторы отмены (отмена предыдущих)**

### **switchMap** — только последний

```javascript
input.valueChanges
  .pipe(
    switchMap((query) => this.searchService.search(query)), // ✅ Только последний поиск!
  )
  .subscribe((results) => {
    /* ... */
  });
```

### **mergeMap** — все параллельно

```javascript
input.valueChanges
  .pipe(
    mergeMap((query) => this.searchService.search(query)), // Все поиски!
  )
  .subscribe((results) => {
    /* ... */
  });
```

## 📊 **7. Обработка ошибок**

```javascript
ajax("/api/users")
  .pipe(
    retry(3), // 3 попытки
    catchError((err) => {
      console.error("API Error:", err);
      return of([]); // Fallback
    }),
  )
  .subscribe({
    next: (users) => console.log(users),
    error: (err) => console.error("Unhandled:", err), // Финальная ошибка
  });
```

## 🎯 **8. Горячие vs Холодные Observable**

```
Холодный (Cold): создаётся при каждой подписке
  interval(1000).subscribe()  // 0,1,2 с начала

Горячий (Hot): общий поток для всех подписчиков
  const hot$ = new Subject();
  hot$.next(1);  // Все подписчики получат!
```

## 🛠️ **Практический пример: Поиск**

```typescript
// Angular компонент
export class SearchComponent {
  @ViewChild("searchInput") input!: ElementRef;

  ngAfterViewInit() {
    fromEvent(this.input.nativeElement, "input")
      .pipe(
        map((e: any) => e.target.value),
        debounceTime(300),
        distinctUntilChanged(),
        switchMap((term) => this.search(term)),
      )
      .subscribe((results) => {
        this.results = results;
      });
  }
}
```

## 🚨 **Типовые ошибки**

```javascript
// ❌ НЕТ подписки = ничего не выполнится!
of(1, 2, 3).pipe(map((x) => x * 2)); // Ленивый!

// ✅ Подписка = выполнение
of(1, 2, 3)
  .pipe(map((x) => x * 2))
  .subscribe(console.log);

// ❌ Забыли отписаться = утечка памяти!
const sub = interval(1000).subscribe();
```


**RxJS = Excel для асинхронных данных**. Цепочка операторов = **декларативная** обработка потоков! 🎯
