# Pick

```
type MyPick<T, K extends keyof T> = {
    [U in K]: T[U];
}
type X = { a: string, b: number, c: boolean };
type PickedX = MyPick<X, 'a' | 'c'>;
const x: PickedX = { a: 'sdsds', c: true };
```
