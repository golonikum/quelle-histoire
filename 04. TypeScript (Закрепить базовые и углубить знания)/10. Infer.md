# infer

**`infer`** — это ключевое слово TypeScript, которое **"достает" типы из других типов** внутри условных типов (`T extends U ? X : Y`). Это как "типовая магия" для создания универсальных утилит.

## 🎯 **Базовый принцип**

```
T extends SomeType<infer R> ? R : never
```

**infer R** говорит TS: "найди тип внутри `SomeType` и назови его `R`".

## 🧠 **1. Извлечение Return Type (ReturnType)**

```ts
type MyReturnType<T> = T extends (...args: any[]) => infer R ? R : never;

const fn1 = () => "hello";
const fn2 = (x: number) => true;

type T1 = MyReturnType<typeof fn1>; // string ✅
type T2 = MyReturnType<typeof fn2>; // boolean ✅
```

**Как работает**:

```
typeof fn1 = () => string
T extends (...args: any[]) => infer R
→ infer R = string
```

## 🔥 **2. Элементы массива**

```ts
type UnwrapArray<T> = T extends (infer U)[] ? U : T;

type T1 = UnwrapArray<string[]>; // string ✅
type T2 = UnwrapArray<number[]>; // number ✅
type T3 = UnwrapArray<{ a: 1 }>; // {a:1} ✅
```

**Разбор**:

```
string[] extends (infer U)[]
→ infer U = string
→ возвращаем string
```

## 🛠️ **3. Promise (Awaited)**

```ts
type UnwrapPromise<T> = T extends Promise<infer U> ? U : T;

type T1 = UnwrapPromise<Promise<string>>; // string ✅
type T2 = UnwrapPromise<Promise<Promise<number>>>; // number ✅
type T3 = UnwrapPromise<number>; // number ✅
```

## 🎯 **4. Параметры функции**

```ts
type MyParameters<T> = T extends (...args: infer Args) => any ? Args : never;

const fn = (name: string, age: number) => {};

type Args = MyParameters<typeof fn>; // [name: string, age: number] ✅
```

## 🚀 **Продвинутые примеры (мидл+)**

### **First letter строки** (собеседование 🔥)

```ts
type FirstLetter<T extends string> = T extends `${infer First}${string}`
  ? First
  : never;

type T1 = FirstLetter<"hello">; // "h" ✅
type T2 = FirstLetter<"a">; // "a" ✅
```

### **Первые N символов**

```ts
type FirstN<T extends string, N extends number> = N extends 0
  ? ""
  : T extends `${infer First}${infer Rest}`
    ? First extends ""
      ? ""
      : `${First}${FirstN<Rest, Subtract<N, 1>>}`
    : "";

type T1 = FirstN<"hello", 3>; // "hel" ✅
```

## 🧠 **Дистрибутивность + infer**

```ts
type IsString<T> = T extends string ? true : false;
type R1 = IsString<string | number>; // true | false ❌

// Но с infer работает по-другому!
type UnwrapUnion<T> = T extends any ? T : never;
type R2 = UnwrapUnion<string | number>; // string | number ✅
```

## 🎯 **React примеры**

### **Props без children**

```ts
type WithoutChildren<T> = T extends React.PropsWithChildren<infer P> ? P : T;

type ButtonProps = WithoutChildren<
  React.ButtonHTMLAttributes<HTMLButtonElement>
>;
// ButtonProps без children ✅
```

### **Хук с автоматическим типом**

```ts
function useStorage<T>(key: string, initial: T): [T, (val: T) => void] {
  // T выводится автоматически из initial
}
```

## 🚨 **Типовые ОШИБКИ**

```ts
// ❌ НЕ РАБОТАЕТ
type Bad<T> = T extends Promise<string> ? infer U : never; // SyntaxError

// ✅ ПРАВИЛЬНО
type Good<T> = T extends Promise<infer U> ? U : never;
```

## 🧪 **Собеседование: напишите ReturnType**

```ts
type MyReturnType<T> = T extends (...args: any[]) => infer R ? R : any; // не функция

type T1 = MyReturnType<() => string>; // string
type T2 = MyReturnType<(x: number) => boolean>; // boolean
type T3 = MyReturnType<string>; // any
```

## 📋 **Шпаргалка паттернов**

```
#1 Функция:     (...args: any[]) => infer R
#2 Массив:      (infer U)[]
#3 Promise:     Promise<infer U>
#8 Template:    `${infer First}${string}`
```

**Запомни**: `infer` **достает тип из конструкции**, `extends` **проверяет**. Вместе = магия TS!

```
Promise<string> ─infer→ string
string[] ─infer→ string
() => string ─infer→ string
```
